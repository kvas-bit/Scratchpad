# ScratchPad

`ScratchPad` is a small macOS menu bar app for saving quick text snippets.

## What It Does

- Opens from the menu bar (no Dock icon)
- Lets you type and save short notes/snippets
- Copy or delete saved snippets
- Saves snippets between launches using `UserDefaults`

## Controls

- `Save` button
- `Cmd + Return` to save
- Copy icon to copy a snippet
- Trash icon to delete a snippet

## Run

```bash
swift build
swift run
```
